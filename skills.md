# Skills Matrix (CMS)

This document summarizes the skills and knowledge areas helpful for maintaining and extending the Clinic Management System.

## Core Backend Skills

- Django 5.x (models, views, templates, auth, admin)
- Django REST Framework (serializers, JSON endpoints)
- Relational databases (SQLite/PostgreSQL)
- Query optimization and pagination
- Data modeling for healthcare workflows

## Frontend Skills

- Bootstrap 5 RTL
- DataTables integration
- SweetAlert2 dialogs
- Vanilla JavaScript for UI behavior
- RTL-friendly layout and typography

## DevOps & Deployment

- Gunicorn application server
- WhiteNoise static file serving
- Railway (DATABASE_PUBLIC_URL configuration)
- Static collection with collectstatic

## Quality & Maintenance

- Writing Django tests
- Managing migrations
- Custom management commands
- Debugging authentication/session issues

## Domain Understanding

- Clinics and branches
- Doctor availability and scheduling
- Appointments and invoices
- Role-based workflows (Manager/Admin/Secretary)
